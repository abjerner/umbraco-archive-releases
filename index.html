[Unofficial] Umbraco Archive Releases

This archive of Umbraco releases is unofficial and has no way been endorsed by Umbraco HQ. Use of the archive is done without any support or warranty.

The sole purpose of this collection is to assist developers with upgrading from a legacy version to the latest release.

Special thanks to Peter Dijksterhuis for supplying the initial archive releases from v2.1.3 to v4.0.3. #h5yr

Latest Release

If you are looking for the latest release of Umbraco, please go to the download page on CodePlex.

    Upgrade Paths

    The general recommended upgrade path is to go from the last patch release up to the next major version release:

    v2.1.x » v3.0.0 » v3.0.6 » v4.0.1 » v4.0.4.2
              » v4.5.0 » v4.5.2 » v4.6.1 » v4.7.0 » v4.7.2
              » v4.8.1 » v4.9.0
              » v4.10.1 » v4.11.4 » v6.0.0 (apply the Path Fixup patch)
